1、
GROUP_CONCAT有个最大长度的限制，超过最大长度就会被截断掉，你可以通过下面的语句获得该长度：
SELECT @@global.group_concat_max_len;
通过以下语句设置想要的最大长度：
SET GLOBAL group_concat_max_len=102400;
